# Reporte de uso de IA - TP TDD POO2

## Herramienta utilizada
Claude (Anthropic) - claude.ai

## Como se uso
Se utilizó IA como asistente durante el desarrollo del TP. La IA sugería el código de cada ciclo TDD (test y luego implementación), el alumno lo revisaba y lo refactorizaba.

## Ciclos asistidos
Los 15 ciclos TDD fueron desarrollados con asistencia de IA, siguiendo siempre el flujo Red → Green → Refactor.

## Decisiones propias del alumno
- Elección de agrupar ciclos en commits para mantener un historial coherente
- Verificación manual de cada test antes de commitear
- Revisión y comprensión del código antes de aceptarlo

## Aclaracion
El uso de IA fue informado y autorizado según las condiciones del trabajo práctico. Todo el código fue revisado y comprendido por el alumno antes de ser commiteado.
